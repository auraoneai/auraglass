## Supported Release Scope

The supported public release scope remains the npm library/package:

- React/Next.js Liquid Glass components, tokens, recipes, styles, workers, package exports, CLI, and optional self-hosted runtime contracts.
- Optional self-hosted AI/WebSocket runtime contracts are documented as infrastructure that consumers can run themselves.
- The legacy demo/mock API server is not a supported production hosted API.
- Live hosted provider, Sentry, docs-hosting, and runtime deployment claims require separate environment evidence.

## Explicit Non-Claims

Do not claim any of the following from the current repository-local evidence alone:

- Full manual accessibility certification.
- Physical phone or tablet touch certification.
- Hosted docs/Storybook deployment proof.
- Live hosted runtime provider/Sentry proof.
- Collaborative document editing support.

## Release Notes Review Result

The checked-in release notes and README scope are consistent with the supported package/library release boundary. The `3.4.x` changelog describes package-surface fixes and does not claim final manual accessibility certification, physical-device certification, live hosted runtime deployment, or hosted docs deployment.

GitHub tagging and final public release approval remain external maintainer actions.
