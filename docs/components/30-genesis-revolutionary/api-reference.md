# Genesis Revolutionary API Reference

This page indexes the Genesis revolutionary component APIs documented in this section.

## Components

- `GlassA11y`: accessibility control and validation surface.
- `CollaborativeGlassWorkspace`: realtime collaborative workspace shell.
- `HoudiniGlassProvider`: CSS Houdini provider for accelerated glass effects.
- `VoiceGlassControl`: voice interaction and command surface.
