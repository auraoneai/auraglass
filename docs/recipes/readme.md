# AuraGlass 3.2 Recipes

The package recipe registry is exported from `aura-glass/registry` and scaffolded through the `aura-glass` CLI.

AuraGlass 3.2 recipes are app-surface starters, not placeholder preview cards. They are expected to scaffold without MUI, Radix, Lucide, or shadcn/ui for core UI, use AuraGlass icons and primitive-backed components, and include accessibility and performance notes.

## Commands

```bash
npx aura-glass list
npx aura-glass info saas-admin-shell
npx aura-glass add saas-admin-shell --out src/components/auraglass/recipes
```
