# Liquid Glass Documentation

This is the dedicated home for AuraGlass Liquid Glass documentation. Use it for material rules, migration guidance, primitive APIs, the release showcase, and the map from existing glass components to newer Liquid Glass patterns.

Component API pages remain in [docs/components/35-liquid-glass](../components/35-liquid-glass/) so developers can browse Liquid Glass inside the normal component catalog.
