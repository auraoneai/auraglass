# Liquid Glass Component Map

| Existing component | Liquid Glass path | Action |
| --- | --- | --- |
| `GlassToolbar` | `LiquidGlassToolbar` | Prefer new component for grouped floating toolbars. |
| `GlassSidebar` | `LiquidGlassInsetSidebar` | Prefer new component for inset split views. |
| `GlassTabBar` | `LiquidGlassTabBar` | Prefer new component for minimizing tab bars and bottom accessories. |
| `GlassSegmentedControl` | `LiquidGlassSegmentedControl` | Prefer new component for shared material selection controls. |
| `GlassBottomSheet`, `GlassDrawer`, `GlassModal` | `LiquidGlassAdaptiveSheet` | Use for adaptive source-aware presentations. |
| `GlassPopover` | `LiquidGlassPopoverMenu` | Use for source-anchored menus. |
| `GlassButton` | `LiquidGlassButtonStyle` or `material="liquid"` | Use new style for prominent Liquid Glass buttons. |
| `GlassCommandPalette`, `GlassSpotlightSearch` | `LiquidGlassCommandSurface` | Use for unified command/search surfaces. |
| Media players | `LiquidGlassMediaControls`, `LiquidGlassNowPlayingBar` | Use clear variant over media with local dimming. |
