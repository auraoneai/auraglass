# LiquidGlassEffectGroup

`LiquidGlassEffectGroup` groups related Liquid Glass controls into one shared material context. Use it for toolbar groups, segmented controls, badge clusters, and floating control clusters.

Checklist:

- Use for related controls that visually belong together.
- Avoid direct glass-on-glass stacking outside an effect group.
- Keep grouped controls keyboard accessible.
- Reduced motion disables morphing but keeps the shared grouping.
